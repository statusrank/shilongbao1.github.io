<h3>问题:</h3>

输入一个链表，反转链表后，输出链表的所有元素。
<!-- more -->

<h3>解答:</h3>

这里我们要求不另外开辟空间将链表反转。

我们只需要开辟一个变量保存下当前结点的上一结点,每次将该点的next修改为指向上一个结点即可,注意提前保存该点的next的值,否则整个链表将失去指向性。

```
class Solution {
public:
    ListNode* ReverseList(ListNode* pHead) {
       //ListNode* rHead = NULL;
       ListNode* pre = NULL;
       ListNode* nxt = NULL;
       while(pHead != NULL)
       {
           nxt = pHead -> next;
           pHead -> next = pre;
           pre = pHead;
           pHead = nxt;
       }
       return pre;
    }
};
```
